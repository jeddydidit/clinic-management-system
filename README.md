 Clinic Management System

📌 Description
This system helps clinics manage  medical stock digitally.
 🛠 Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript

⚙️ Features
- Stock tracking
- Report generation

🗄 Database
Import the `database.sql` file into phpMyAdmin.

🚀 How to Run
1. Install XAMPP
2. Place project in htdocs
3. Start Apache & MySQL
4. Open http://localhost/project

📄 Documentation
See `/docs` folder
